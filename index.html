    Connessione Google Sheets
    Come configurare: Crea un nuovo Google Sheets → Estensioni → Apps Script → incolla lo script → Distribuisci come web app → copia l'URL qui sotto.
    Gestione Dati

    I dati vengono salvati su Google Sheets tramite Apps Script. L'URL dello script viene memorizzato localmente nel browser.

    Guida Apps Script
    1 — CREA IL FOGLIO
    Vai su sheets.google.com → Crea nuovo foglio → Rinominalo "Xstamina Body Monitor"
    2 — AGGIUNGI LO SCRIPT
    Estensioni → Apps Script → Incolla il codice fornito → Salva (Ctrl+S)
    3 — DISTRIBUISCI
    Distribuisci → Nuova distribuzione → Tipo: "App web" → Chi ha accesso: "Chiunque" → Distribuisci → Copia URL
    4 — INCOLLA L'URL QUI
    Incolla l'URL nel campo "Script URL" in cima a questa pagina → Salva
